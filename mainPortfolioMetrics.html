Portfolio Metrics Calculation

Introduction

This page provides a step-by-step guide to manually calculate the key metrics for a portfolio of assets, including PnL, Total PnL, CAGR, and Standard Deviation (Risk).

PnL Calculation

To calculate the Profit and Loss (PnL) for each asset:


PnL = (Ending Price - Starting Price) × Number of Shares

Example:

Starting Price = 100
Ending Price = 120
Number of Shares = 10
Calculation:
PnL = (120 - 100) × 10 = 200

Total PnL Calculation

To calculate the Total PnL for the portfolio:


Total PnL = Σ (PnL for each asset)

Example:

PnL for Asset 1 = 200
PnL for Asset 2 = 100
Calculation:
Total PnL = 200 + 100 = 300

CAGR Calculation

To calculate the Compound Annual Growth Rate (CAGR):


CAGR = ((Ending Value / Starting Value) ^ (1 / Number of Years)) - 1

Example:

Starting Value = 1000
Ending Value = 2000
Number of Years = 5
Calculation:
CAGR = ((2000 / 1000) ^ (1 / 5)) - 1 ≈ 0.1487 or 14.87%

Standard Deviation (Risk) Calculation

To calculate the Standard Deviation (Risk) for the portfolio:


σ = sqrt((1 / N) Σ (x_i - μ)²)

Where:

  • N is the number of observations
  • x_i is each individual PnL value
  • μ is the mean of the PnL values

Steps:

  1. Calculate the mean (μ) of the PnL values:
    μ = (Σ x_i) / N
  2. Calculate each squared deviation from the mean and then the variance:
    Variance = (1 / N) Σ (x_i - μ)²
  3. Calculate the standard deviation by taking the square root of the variance:
    σ = sqrt(Variance)

Example:

PnL values: 200, 100
Step 1: Calculate the mean:
μ = (200 + 100) / 2 = 150
Step 2: Calculate squared deviations and variance:
(200 - 150)² = 2500
(100 - 150)² = 2500
Variance = (2500 + 2500) / 2 = 2500
Step 3: Calculate the standard deviation:
σ = sqrt(2500) = 50
